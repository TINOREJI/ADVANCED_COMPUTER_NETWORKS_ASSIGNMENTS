## 📘 About This Assignment

This project demonstrates how to configure **OSPF (Open Shortest Path First)** routing using **Cisco Packet Tracer**.

Unlike static routing or RIP, OSPF uses a **link-state algorithm** and calculates the best path using the **Shortest Path First (SPF)** algorithm.

---

## 🎯 Objective

* Configure OSPF on multiple routers
* Assign router IDs
* Advertise networks using OSPF
* Analyze routing decisions using cost
* Verify routing tables and neighbors

---

## 📊 OSPF Analysis

* OSPF selects path based on **cost**
* Cost depends on bandwidth
* Lower cost = preferred path

Changing bandwidth:

```bash id="ospfcost"
bandwidth 64
```

This affects route selection.

## ✅ Result

OSPF routing was successfully configured, and all routers dynamically exchanged routing information. Communication between networks was achieved using the shortest path.

## 💡 What I Learned
