## 📘 About This Assignment

This assignment focuses on configuring **static routing** between two different networks using **Cisco Packet Tracer**.

The goal is to allow communication between two separate LANs by manually defining routes on routers.



## 🎯 Objective

* Connect two different networks
* Configure router interfaces
* Implement static routing
* Verify communication between hosts



## 🧠 Scenario

Two branch offices (Tehran and Shiraz) are connected using routers. Each branch has its own LAN, and the routers are connected using a serial link.

To enable communication between both LANs, static routes are configured.



## 🛠️ Network Details

| Device      | Network        |
| ----------- | -------------- |
| R1 LAN      | 192.168.1.0/24 |
| R2 LAN      | 172.16.0.0/16  |
| Router Link | 10.0.0.0/30    |



## ⚙️ Configuration Steps

### 🔹 Router R1

```bash
interface fastethernet0/0
ip address 192.168.1.1 255.255.255.0
no shutdown

interface serial1/0
ip address 10.0.0.1 255.255.255.252
clock rate 64000
no shutdown

ip route 172.16.0.0 255.255.0.0 10.0.0.2
```

### 🔹 Router R2

```bash
interface serial1/0
ip address 10.0.0.2 255.255.255.252
no shutdown

interface fastethernet0/0
ip address 172.16.0.1 255.255.0.0
no shutdown

ip route 192.168.1.0 255.255.255.0 10.0.0.1
```
